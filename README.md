# smart-water-jaipur-ai
AI-powered system to predict water tanker demand, identify scarcity zones, and detect price exploitation for Smart City water management.
## Smart Water Jaipur – AI-Based Water Demand & Price Monitoring System

This project addresses urban water scarcity by leveraging data analytics and machine learning to predict water tanker demand, identify high-risk shortage zones, and detect abnormal pricing practices.

### Key Features
- Predicts daily and weekly water tanker demand using machine learning
- Identifies water scarcity zones using clustering techniques
- Detects price exploitation through anomaly detection
- Interactive dashboard built using Streamlit
- Supports Smart City planning, NGOs, and municipal authorities

### Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit
- VS Code

### How to Run
```bash
pip install -r requirements.txt
streamlit run src/dashboard.py
